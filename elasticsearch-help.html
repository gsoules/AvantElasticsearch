Search Tips
You can use the following special characters in the Search Box above and at left in the Keywords field, and in the Contains condition:
  • + before a keyword makes it required
  • - before a keyword excludes it
  • " wraps a phrase to search for
  • | signifies OR
  • * after a keyword signifies a wildcard
  • ( and ) signify precedence
  • To search by Contributor, use the Contributor Id from the table shown here.
